# Expense Tracker in React JS
### Features: 
- Transactions History
- Income/Expense Display
- Balance
- Add Transactions
- Delete Transactions
- Clear Inputs
- Check for empty Fields


### Technology Used:
- React Hooks ( ✔️ useContext, ✔️ useState, ✔️ useReducer, ✔️ useEffect)
